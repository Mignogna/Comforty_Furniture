## E-commerce de móveis

Projeto de e-commerce focado na indústria de móveis, construído com **React.js** e **Vite**.
A estilização foi implementada com **Tailwind CSS**, complementada pela biblioteca de componentes **DaisyUI** para agilizar o desenvolvimento dos botões e pela blblioteca **Lucide-react** para a utilização dos ícones interativos.
A experiência do usuário é aprimorada pelo uso de sliders de imagens, implementados com **React Slick**, que exibem os produtos em ambientes decorados e inspiradores, permitindo que os clientes visualizem as peças em contexto.


## Furniture E-commerce

An e-commerce project focused on the furniture industry, built with **React.js** and **Vite**.
Styling is handled by **Tailwind CSS**, complemented by the **DaisyUI** component library to streamline button development and the **Lucide-react** library for interactive icons.
The user experience is enhanced by image sliders, implemented with **React Slick**, which showcase products in decorated and inspiring settings, allowing customers to visualize the pieces in context.
